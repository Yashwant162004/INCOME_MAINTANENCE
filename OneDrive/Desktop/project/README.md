# Premium Construction Website (React + Tailwind)

Luxury-focused construction company website with modern UI/UX, smooth animations, and responsive design.

## Features
- React + Vite
- Tailwind CSS with dark + gold theme
- Framer Motion page transitions & micro-interactions
- Responsive pages: Home, Projects, Project Detail, About, Services, Testimonials, Contact
- Sticky blurred navbar
- Filterable project gallery
- Testimonials carousel
- Animated global stats
- Before/After slider
- Contact form + Google Maps + WhatsApp CTA
- Consultation modal + loading screen

## Run
1. `npm install`
2. `npm run dev`
3. Open `http://localhost:5173`

## Build
- `npm run build`
- `npm run preview`

## Deployment
Deploy on Vercel/Netlify: select project folder and set build command `npm run build`, publish directory `dist`.
